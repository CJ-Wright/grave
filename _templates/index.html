Dead simple graph visualization

Grave is a graph visualization package combining ideas from Matplotlib, NetworkX, and seaborn. Its goal is to provide a network drawing API that covers the most use cases with sensible defaults and simple style configuration.

Features

  • Supports drawing graphs from NetworkX
  • Open source 3-clause BSD license
